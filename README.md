Uma plataforma web de ensino online desenvolvida para oferecer uma experiência de aprendizado moderna, intuitiva e altamente organizada. A aplicação possui uma interface visual dinâmica, com seções de cursos organizadas de forma clara, recomendações inteligentes e fácil exploração de conteúdos.

A plataforma conta com três áreas principais:

Painel Administrativo (Dashboard) Os administradores possuem acesso a um dashboard informativo com métricas detalhadas da plataforma, como número de alunos ativos, cursos publicados, taxa de conclusão, crescimento de usuários e desempenho financeiro. O painel permite gerenciar cursos, usuários, professores, categorias e acompanhar estatísticas em tempo real.

Área dos Professores Os professores podem criar, editar e publicar cursos, organizar módulos e aulas, adicionar vídeos e materiais complementares, além de acompanhar o progresso dos alunos inscritos. A interface também permite visualizar avaliações, comentários e estatísticas de engajamento de cada curso.

Área dos Alunos Os alunos possuem uma página personalizada onde podem explorar cursos, assistir aulas, acompanhar seu progresso, salvar cursos favoritos e receber recomendações baseadas em seus interesses. O layout foi pensado para facilitar a navegação e tornar o processo de aprendizado mais envolvente e acessível.

A plataforma foi projetada para ser responsiva, escalável e centrada na experiência do usuário, oferecendo uma estrutura completa para gestão e consumo de conteúdos educacionais online.

An online learning web platform designed to provide a modern, intuitive, and highly organized learning experience. The application features a dynamic interface with clearly organized course sections, smart recommendations, and easy content discovery.

The platform includes three main areas:

Administrative Dashboard Administrators have access to an informative dashboard with detailed platform metrics such as active students, published courses, completion rates, user growth, and financial performance. The dashboard also allows management of courses, users, instructors, categories, and real-time statistics monitoring.

Instructor Area Instructors can create, edit, and publish courses, organize modules and lessons, upload videos and additional materials, and track the progress of enrolled students. The interface also provides access to reviews, comments, and engagement analytics for each course.

Student Area Students have a personalized page where they can explore courses, watch lessons, track their learning progress, save favorite courses, and receive recommendations based on their interests. The layout is designed to make navigation simple and create a more engaging and accessible learning process.

The platform is built to be responsive, scalable, and user-focused, providing a complete structure for managing and consuming online educational content.
